Yuki é um bot de WhatsApp que oferece downloads, comandos de administração e funções de entretenimento para grupos. Rápido, confiável e fácil de usar!

> Este projeto é livre para alterações no código, mas não é permitida a venda do mesmo. Ele é gratuito e pode ser utilizado por qualquer pessoa. No entanto deixe créditos ao [Speed](https://github.com/speedxzpdev) e [Lenoz7](https://github.com/LuisAlves10)

## Mais informações:
<details>
<summary>Instalação</summary>

**Instalação:**
> O comando pode variar de ambiente

**Requisitos:**

- node 18+(recomendado)
- Git 
- WhatsApp
- ffmpeg

**Instalação do node:**
```bash
pkg install nodejs
```

**Instalação do git:**
```bash
pkg install git
```

**Clonagem do repositório:**
```bash 
git clone "https://github.com/speedxzpdev/YukiV3"
```
</details>


<details>
<summary>Configuração💖</summary>
Ao clonar copie o .env.example e configure seu .env

você pode mudar algumas configuração da bot pelo config.js como prefixo, nome e afins.

Para que consiga usar comandos de download e Ai compre uma api key na [ZeroTwoApi](https://zero-two-apis.com.br/).
</details>

<details>
<summary>Estrutura de pastas👻</summary>

**src/config.js**
> Arquivo básico de configuração. Nele há variaveis como **nome**, **prefix** e etc.

**src/server.js**
> Servidor http que escuta webhooks de pagamento

**src/events**
> Aqui fica eventos do baileys e outros criados pelo poprio bot

**src/commands**
> Aqui fica todos os comandos

**src/database**
> Pastas que guardam models do mongoose e alguns json.

**src/assets**
> Aqui fica dados de sessão, fotos e arquivos temporários.

**src/lib**
> Aqui fica todas as instâncias de algumas lib como mercado pago, axios, mongoDB.

**src/utils**
> Aqui fica funções utilitarias como tiktok.js, instagram.js e outras coisas.



</details>
